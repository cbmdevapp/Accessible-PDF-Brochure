Adds an accessible PDF brochure section to WooCommerce single product pages with modern responsive design and enhanced accessibility features.

== Description ==

This plugin adds an accessible tab to WooCommerce single product pages where a product brochure (PDF file) can be displayed or downloaded. The tab is designed with accessibility in mind for visually impaired users and follows modern web standards.

**Features:**
- Modern object-oriented PHP architecture
- Enhanced ARIA roles and accessibility features
- Responsive design for all devices
- Modern CSS with gradients and animations
- Performance optimizations
- Error handling and graceful degradation
- Internationalization support
- WooCommerce compatibility checks
- Modern JavaScript with ES6+ features
- High contrast mode support
- Reduced motion support
- Focus management for keyboard navigation

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to your product edit page and add a PDF brochure URL in the "PDF Brochure URL" field.

== Frequently Asked Questions ==

= How do I upload a PDF for a product? =
Go to your product edit page in WordPress admin, scroll down to the "Product Data" section, and enter the PDF file URL in the "PDF Brochure URL" field.

= Can I translate the plugin? =
Yes! Use the `.pot` file included in the `languages/` directory for translation. The plugin supports full internationalization.

= Does this plugin work with WooCommerce 8.x? =
Yes! This plugin is fully compatible with WooCommerce 8.x and has been tested up to version 8.5.

= Is this plugin accessible? =
Yes! The plugin follows WCAG 2.1 AA guidelines and includes proper ARIA attributes, keyboard navigation support, and screen reader compatibility.
