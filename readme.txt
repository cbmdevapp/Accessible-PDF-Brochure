Adds an accessible PDF brochure section to WooCommerce single product pages with responsive design.

== Description ==

This plugin adds an accessible tab to WooCommerce single product pages where a product brochure (PDF file) can be displayed or downloaded. The tab is designed with accessibility in mind for visually impaired users.

**Features:**
- ARIA roles for accessibility.
- Responsive design for mobile devices.
- Tab positioned next to the "Description" tab.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Add a custom field `_product_brochure` to your product and input the PDF file URL.

== Frequently Asked Questions ==

= How do I upload a PDF for a product? =
Use the custom field `_product_brochure` to input the URL of the PDF file for the product.

= Can I translate the plugin? =
Yes! Use the `.pot` file included in the `languages/` directory for translation.
